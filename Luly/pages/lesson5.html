I can talk to robots!

Vocabulary | الكلمات

Coding Missions | مهمات البرمجة

1. A robot needs a _____ to work.
يحتاج الروبوت إلى _____ لكي يعمل.
Battery
Apple
2. We speak to robots using _____.
نتحدث إلى الروبوتات باستخدام _____.
Code
Music
3. "Move Left" is a _____.
جملة "تحرك يساراً" تعتبر _____.
Command
Game

Final Script | العرض النهائي

اشرحي كيف تبرمجين صديقك الآلي!

English:

"Hello! I have a Robot friend. I give him a Command using a special Code. My robot needs a Battery for energy. I am a smart programmer. Thank you!"
العربي:
"مرحباً! لدي صديق روبوت. أعطيه أمراً باستخدام كود خاص. روبوتي يحتاج بطارية للحصول على الطاقة. أنا مبرمجة ذكية. شكراً لكم!"
